## Genuine Assets

Before reviewing claim families, the following assets have been independently verified:

1. **58 verified CP2K DFT calculations** with cloud provenance (Inductiva task IDs). These compute binding energies for Janus Ligand-metal complexes at the PBE/DZVP-MOLOPT-SR-GTH level (no dispersion correction applied). All 58 converged and are forensically auditable.
2. **Kremser separation framework** with 18-point sensitivity analysis. The Kremser equation is rigorous first-principles thermodynamics (not an empirical fit). The sensitivity sweep shows defensible CapEx reduction of 20-40% depending on practical correction factor.
3. **405 RDKit-generated scaffolds** in SDF format (`expanded_candidates.sdf`), all validated with 3D coordinates and chemical sanitization.
4. **Analytical Born model for pore selectivity screening** -- correct continuum electrostatics for ion dehydration barriers. Produces physically meaningful Li+/Na+ selectivity predictions.

### Caveats on Frequently Overstated Claims

- **"DFT-calibrated" binding energies:** Only 58 real CP2K DFT calculations exist. The remaining ~166 "expanded" data points are generated by a calibrated physics formula (analytical Coulomb+LJ+Born), NOT quantum DFT. Furthermore, the 58 DFT runs use La (lanthanum) as a proxy for Nd (neodymium) because all Nd CP2K runs aborted. La and Nd have similar ionic radii but different f-electron configurations; the systematic error from this substitution is unknown.
- **ML surrogate model:** Trained on 58 real + 166 synthetic points using one-hot and molecular fingerprint encoding. The v8 model (Ridge R-squared = 0.966) improves on v7 but the one-hot component means it functions partly as a lookup table, not a generalizing predictor. It has no demonstrated predictive power for unseen scaffold families.
- **GROMACS molecular dynamics:** Only 10 ps energy minimization (steepest descent) was completed. This is NOT production MD. No equilibration run, no production trajectory, no statistically converged properties were obtained.
- **Scaffold count:** The expanded candidate library (`expanded_candidates.sdf`) contains 405 molecules, not 730. The original `candidates.sdf` contains 730 Janus Ligand structures (predominantly systematic alkyl tail permutations across 4 scaffold families).

## Prior Art Differentiation

The Fluorocatcher claims are differentiated from prior art in PFAS sorbent technology by:

1. **Dual-mechanism binding:** No prior art sorbent simultaneously engages both the anionic head group (electrostatic) and perfluoroalkyl tail (fluorous affinity) of PFAS compounds.

2. **Irreversibility threshold:** No prior art sorbent achieves binding energies below -80 kJ/mol for PFAS compounds. GAC, IX, and RO all operate above this threshold. (Note: this threshold is a computational prediction from analytical models, not experimental measurement.)

3. **Computational design methodology:** The integrated pipeline (combinatorial generation + analytical binding estimates + ML screening) represents a novel method of discovering PFAS sorbents that is not described in prior art.

4. **Scaffold library breadth:** The 405-scaffold expanded library provides compositional breadth that cannot be designed around without infringing one or more of the dependent claims covering cation types, spacer lengths, fluorination levels, and topologies.
